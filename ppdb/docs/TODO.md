# PPDB 开发进度

## 已完成功能

### 1. 基础组件
- [x] 引用计数（`ref_count`）
  - [x] 原子操作支持
  - [x] 自定义释放函数
  - [x] 线程安全的内存管理

### 2. 跳表实现
- [x] 有锁跳表（`skiplist`）
  - [x] 基本的增删改查操作
  - [x] 迭代器支持
  - [x] 线程安全（互斥锁）
- [x] 无锁跳表（`atomic_skiplist`）
  - [x] 无锁并发操作
  - [x] 原子状态管理
  - [x] 引用计数内存管理

### 3. 内存表实现
- [x] 基本内存表（`memtable`）
  - [x] 基于跳表的实现
  - [x] 大小限制管理
  - [x] 线程安全操作
- [x] 分片内存表（`sharded_memtable`）
  - [x] 基于无锁跳表
  - [x] 分片策略
  - [x] 并发性能优化

### 4. KVStore 核心
- [x] 基本框架
  - [x] 有锁/无锁模式支持
  - [x] 错误处理
  - [x] 日志系统
